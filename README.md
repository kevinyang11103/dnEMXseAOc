# 小学生成绩管理系统 python390

## 项目概述

小学生成绩管理系统是一款基于Python语言开发，用于辅助学校教师进行学生成绩管理、查询和分析的软件。

## 技术栈

- **后端**：Python
- **数据库**：MYSQL
- **前端**：PyQt5

## 运行环境

- **开发环境**：PyCharm

## 功能模块

- 学生信息管理
  - 学生信息录入
  - 学生信息编辑
  - 学生信息查询
- 成绩管理
  - 成绩录入
  - 成绩修改
  - 成绩查询
- 成绩统计与分析
  - 班级成绩分布图
  - 学生成绩趋势图

## 系统角色

- 管理员：负责系统的维护和用户管理
- 教师：负责学生信息管理、成绩录入与查询

## 目录结构

```plaintext
.
├── src/
│   ├── main.py          # 主程序入口
│   ├── models.py        # 数据模型定义
│   ├── views.py         # 视图与界面逻辑
│   ├── controllers.py   # 控制器逻辑
│   └── utils.py         # 工具模块
├── database/            # 数据库相关文件
└── ui/                  # 界面设计相关文件
```

## 核心亮点

- **友好的用户界面**：采用PyQt5构建，界面友好，操作简单。
- **数据分析功能**：提供直观的图表分析，便于了解学生的学习状况。
- **灵活的数据管理**：支持学生信息的增删改查，以及成绩的快速录入与修改。
- **安全性**：使用MYSQL数据库保证数据存储的安全可靠。
